# Vilmos - Solutions Digitales & Sécurité High-Tech

Ce projet est le site web officiel de Vilmos, une entreprise basée à Conakry, Guinée, spécialisée dans le développement d'applications sur-mesure et les solutions de vidéosurveillance intelligentes.

## Structure du Projet

- `index.html` : La page d'accueil principale du site.
- `assets/css/style.css` : Styles CSS personnalisés (animations, scrollbar, etc.).
- `assets/js/script.js` : Scripts JavaScript pour l'interactivité (thème sombre, estimateur de budget, simulateur jour/nuit).

## Fonctionnalités Clés

1. **Estimateur de Devis en Temps Réel** : Permet aux clients de configurer leurs besoins et d'obtenir une estimation instantanée en GNF et en Euros.
2. **Simulateur de Vision Nocturne** : Une démonstration interactive de la technologie de surveillance infrarouge.
3. **Design Responsive** : Optimisé pour mobile, tablette et desktop.
4. **Mode Sombre/Clair** : Basculez entre deux ambiances visuelles en un clic.

## Installation Local

Pour visualiser le projet localement, ouvrez simplement le fichier `index.html` dans votre navigateur ou utilisez une extension de serveur local (comme "Live Server" dans VS Code).
